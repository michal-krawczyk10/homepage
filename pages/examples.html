Przykłady różnych rozwiązań

stopka

Żeby stopka została na dole strony, tag main powinien dostać klasę

min-height: calc(100vh - (wysokość stopki + wysokość nagłówka + ew. margin));

w ten sposób, stopka zawsze będzie na namym dole, bez względu na ilość treści na stronie

funkcja wrzucająca content na stronę

funkcja najpierw wybiera z html element wskazany atrybutem, a następnie za pomocą innerHTML wrzuca wskazany kontent we wskazanym elemencie

function createContent(querySelectorContent, content) {
const element = document.querySelector(querySelectorContent);
element.innerHTML = content;
}

wywołując ją, nadajemy jako pierwszy atrybut rządany element (np. jakąś klasę), jako drugi oczekiwany kontent

createContent( ".welcome__header--sub--js", `tekst, który pojawi się we wskazanym miejscu`);

dzisiejsza data za pomocą js

najpierw tworzymy datę i definiujemy jej format

let date = new Date();
let day = date.getDate();
let month = date.getMonth() + 1;
let year = date.getFullYear();
let today = day + '/' + month + '/' + year

potem wrzucamy ją na stronę np. wykorzystując opisaną wcześniej funkcję do podmiany treści

createContent(".jakas__klasa--js", '"dziś jest ${today}';
tu będzie jakaś stopka
