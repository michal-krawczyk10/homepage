Tu będzie magia js'a, którego jak narazie nie kumam.

Typy danych

zmienna

Zmienna to nie jest wartość, zmienna tylko wskazuje na jakąś wartość!

let vs const vs var
var jest uniwersalne, obecnie się od niej odchodzi;
const to zmienna, które ma stałe połączenie z wartością i się nie zmienia (od constant);
let - zmienna, której wartość można zmienić

Zmienną definiuje się tylko raz. W zmiennej może być cokolwiek.

Typy proste w JS (primitives)

String - tekst, otaczany 'apostrofami'

number - wiadomo

boolean - wartość: true lub false

null - wartość niezdefiniowana celowo, która gdzieś później się pojawi, lub musimy zaznaczyć że ta zmienna istnieje, ale nie znamy jej wartości. Wskazuje na 'nic'

undefined - zmienna niezdefiniowana, do której się próbujemy dostać

Można sprawdzić w konsoli, jakiego typu jest zmienna, za pomocą wyrażenia, przydaje się do sprawdzenia czy zmienna jest zdefiniowana
typeof nazwaZmiennej

immutability - 'niezmienność', 'niemutowalność'

Nie możemy zmieniac wartości primitive values w JS, wartości zmiennych istnieją same w sobie, a my tylko zmieniamy połączenia między nimi i do nich.
let x = 10, nie oznacza, że x jest równe 10, tylko, że x wskazuje na 10. 10 istnieje niezależnie od x.

DOM - Document Object Model

DOM to "złożony w przeglądarce HTML"
JS może go modyfikować.
funkcja querySelector zwraca node (węzeł) z Html-a, selektor zapisywany jest zgodnie z css
korzystając z BEM - tworzymy dodatkową klasę z modyfikatorem --js, ułatwia to identyfikację i usuwa potencjalne problemy

Metody JS

Podpięcie DOM do html / JS do elementu HTML

aby podpięć js pod wybrany element html, można skorzystać m.in. z metody querySelector

const welcomeHeader = document.querySelector('welcome__header--js');

W ten sposób przypisujemy zmiennej welcomeHeading, element html wskazany przy pomocy klasy welcome__header--js.

Tworzenie klasy z modyfikatorem --js, zapobiega zgubieniu podpięcia gdy zmienimy coś w htmlu. Klasy tej nie należy stylować, służy tylko wskazaniu elementu js'owi.

Z tak podpiętym elementem, możemy robić różne rzeczy za pomocą JS

innerHTML

Za pomocą tej metody, możemy podmienić treść wskazanego wcześniej elementu.

welcomeHeader.innerHTML = 'Miło że wpadłaś / wpadłeś :)';

Ta komenda dodaje na stronę podany string, w podpiętym wyżej miejscu, poprzez przypisanie zmiennej nowej wartości, widocznej w DOM ale nie w HTML
